# DukesCleanSteamUI

Steam skin, removing the following elements from the client UI

- Homepage:
  - What's new section
- Game detail page:
  - Survey section ('Would you like to recommend ...')
  - Activity input box
- Titlebar:
  - Steam wallet account balance (available in drop-down still)
